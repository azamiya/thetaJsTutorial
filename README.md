# THETA JS Tutorials #

### What is this repository for? 
This tutorial is to transmit 360 video from RICOH THETA via peerJS.

### How do I get set up? ###

```sh
$ git clone https://github.com/cdra0624/thetaJsTutorial.gitcreate2LocalController.git
$ cd thetaJsTutorial
$ npm install
$ npm start
```
